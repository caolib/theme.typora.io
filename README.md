## 1.宽度适应文本

> 引用块的宽度适应文本

> [!tip]
>
> gh警告框也是

```css
// 代码块宽度也是
#write blockquote:hover {
    background: #fff;
    box-shadow: 0 8px 15px rgba(0, 0, 0, .3);
    transform: var(--transform);
}
```
